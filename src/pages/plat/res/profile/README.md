## 资源档案管理结构 （pages/plat/res/profile）

```text
/profile/
  ├── index                       # 资源列表
  ├── component                   # 资源编辑tab页内容
  │   ├── BasicInfo                 # 基本信息
  │   ├── Cert                      # 资质证书
  │   ├── Edubg                     # 教育经历
  │   ├── Finance                   # 财务信息
  │   ├── Getrp                     # 奖惩信息
  │   ├── PlatInfo                  # 平台信息
  │   ├── Workbg                    # 工作经历
  ├── config                      # 资源详情页tab页的设置
  ├── modal                       # 资源新增编辑的弹出框
  │   ├── index                     # export
  │   ├── AbAccModal                # 财务信息弹框
  │   ├── CertModal                 # 资质证书弹框
  │   ├── EdubgModal                # 教育经历弹框
  │   ├── GetrpModal                # 奖惩信息弹框
  │   ├── WorkbgModal               # 工作经历弹框
  ├── BackgroundInfo              # 资源新增-背景信息
  ├── CapaInfo                    # 资源新增-能力信息
  │   ├── CapaTreeTransfer          # 资源新增-能力树
  │   ├── CapasetTreeTransfer       # 资源新增-复合能力树
  ├── Detail                      # 资源新增-基本信息
  ├── DetailEdit                  # 资源编辑
  ├── DetailQuery                 # 资源详情
  ├── FinanceInfo                 # 资源新增-财务信息
  └── PlatInfo                    # 资源新增-平台信息
```
